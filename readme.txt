Duplicate Post or Page with it's all the contents in just single click.

== Description ==

Duplicate WP Post Page Clone is a plugin that allows user to duplicate or replicate post,page and different post type in just single click.

WP Post Page Clone Plugin Features

* Post replicate with it's all the contents in just single click.
* Page replicate with it's all the contents in just single click.
* Cross Browsers Support ( Chrome, Safari, Firefox, etc. )

== Installation ==

* Clone the directory into your plugins folder of Wordpress site or install the zipped directory of this plugin in the plugin page.
* Activate the plugin through the 'Plugins' page in WordPress

== Frequently Asked Questions ==

= How to replicate/duplicate a post ? =

* Activate the plugin from the 'Plugins' page in WordPress
* Now go to list of posts
* Hover your cursor over any post you will see a "Duplicate as draft" link
* Click on that link to replicate/duplicate your post with it's all settings of that post.That will replicate/duplicate as a draft

= How to replicate/duplicate a page ? =

* Activate the plugin from the 'Plugins' page in WordPress
* Now go to list of posts
* Hover your cursor over any post you will see a "Duplicate as draft" link
* Click on that link to replicate/duplicate your page with it's all settings of that page.That will replicate/duplicate as a draft


= Is the plugin work with Custom post type ? =

* Yes, definately the plugin works with the custom post type as well.
